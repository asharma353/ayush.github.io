Anomaly Detection via Netflow & DNS

An open‑source research project that proposes an anomaly detection system to flag DNS poisoning and rogue DNS authorities. The system performs passive DNS analysis over 27 days and clusters domains into benign, static or malicious anomalies based on their network profiles.

  • Queries CYMRU WHOIS data to build a domain profile repository.
  • Clusters domains by comparing them to a baseline (day 0).
  • Classifies domains as benign, static or malicious anomalies.
